# SeCuida

Um santuário digital para educadores. Oferece orientação emocional, recursos de regulação e uma rede de apoio confidencial.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
